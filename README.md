## ❓ Troubleshooting

- Download the latest release again

If Windows says the app is blocked:

- Right-click the file
- Open Properties
- Check Unblock if you see it
- Click Apply
- Try opening the app again

If the window opens and closes at once:

- Make sure your system meets the requirements
- Redownload the release in case the file was not complete
